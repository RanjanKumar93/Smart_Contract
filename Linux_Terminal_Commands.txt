1. pwd
   - Example: pwd
   - Explanation: Prints the current working directory.

2. ls
   - Example: ls
   - Explanation: Lists the files and directories in the current directory.

3. cd
   - Example: cd /home/user/Documents
   - Explanation: Changes the current directory to the specified path.

4. mkdir
   - Example: mkdir new_folder
   - Explanation: Creates a new directory named 'new_folder'.

5. rmdir
   - Example: rmdir old_folder
   - Explanation: Removes an empty directory named 'old_folder'.

6. rm
   - Example: rm file.txt
   - Explanation: Removes the specified file or directory (use -r for directories).

7. cp
   - Example: cp file.txt /home/user/backup/
   - Explanation: Copies a file or directory to a specified location.

8. mv
   - Example: mv old_name.txt new_name.txt
   - Explanation: Moves or renames a file or directory.

9. cat
   - Example: cat file.txt
   - Explanation: Displays the contents of a file.

10. more
    - Example: more file.txt
    - Explanation: Displays file contents one screen at a time.

11. less
    - Example: less file.txt
    - Explanation: Similar to 'more', but with more capabilities like backward navigation.

12. head
    - Example: head -n 5 file.txt
    - Explanation: Displays the first 5 lines of a file.

13. tail
    - Example: tail -n 5 file.txt
    - Explanation: Displays the last 5 lines of a file.

14. touch
    - Example: touch newfile.txt
    - Explanation: Creates an empty file or updates the timestamp of an existing file.

15. chmod
    - Example: chmod 755 script.sh
    - Explanation: Changes the permissions of a file or directory.

16. chown
    - Example: chown user:group file.txt
    - Explanation: Changes the owner and group of a file or directory.

17. find
    - Example: find /home/user/ -name "*.txt"
    - Explanation: Searches for files and directories within a specified path.

18. grep
    - Example: grep "search_term" file.txt
    - Explanation: Searches for a specified string within files.

19. df
    - Example: df -h
    - Explanation: Displays the disk space usage of file systems in a human-readable format.

20. du
    - Example: du -sh /home/user/Documents
    - Explanation: Shows the disk usage of a directory and its contents.

21. ps
    - Example: ps aux
    - Explanation: Displays currently running processes.

22. top
    - Example: top
    - Explanation: Displays real-time system resource usage, including CPU and memory.

23. kill
    - Example: kill 1234
    - Explanation: Terminates a process with the specified process ID (PID).

24. wget
    - Example: wget https://example.com/file.txt
    - Explanation: Downloads files from the web.

25. curl
    - Example: curl -O https://example.com/file.txt
    - Explanation: Transfers data from or to a server, supports various protocols.

26. tar
    - Example: tar -czvf archive.tar.gz /path/to/directory
    - Explanation: Compresses (or extracts) files using tarball format.

27. unzip
    - Example: unzip archive.zip
    - Explanation: Extracts files from a zip archive.

28. zip
    - Example: zip -r archive.zip /path/to/directory
    - Explanation: Compresses files into a zip archive.

29. man
    - Example: man ls
    - Explanation: Displays the manual for a command.

30. echo
    - Example: echo "Hello, World!"
    - Explanation: Outputs the specified string to the terminal.

31. sudo
    - Example: sudo apt-get update
    - Explanation: Runs commands with superuser privileges.

32. apt-get
    - Example: sudo apt-get install package-name
    - Explanation: Manages packages in Debian-based distributions.

33. yum
    - Example: sudo yum install package-name
    - Explanation: Manages packages in Red Hat-based distributions.

34. scp
    - Example: scp file.txt user@remote:/path/to/destination
    - Explanation: Securely copies files between hosts over SSH.

35. ssh
    - Example: ssh user@remote
    - Explanation: Connects to a remote machine over SSH.

36. history
    - Example: history
    - Explanation: Displays the command history.

37. alias
    - Example: alias ll='ls -la'
    - Explanation: Creates a shortcut for a command.

38. nano
    - Example: nano file.txt
    - Explanation: Opens a file in the Nano text editor.

39. vim
    - Example: vim file.txt
    - Explanation: Opens a file in the Vim text editor.

40. passwd
    - Example: passwd
    - Explanation: Changes the user's password.

41. whoami
    - Example: whoami
    - Explanation: Displays the current logged-in username.

42. hostname
    - Example: hostname
    - Explanation: Displays or sets the system's hostname.

43. uname
    - Example: uname -a
    - Explanation: Displays system information.

44. reboot
    - Example: sudo reboot
    - Explanation: Reboots the system.

45. shutdown
    - Example: sudo shutdown now
    - Explanation: Shuts down the system immediately.

46. date
    - Example: date
    - Explanation: Displays the current date and time.

47. cal
    - Example: cal
    - Explanation: Displays a calendar.

48. clear
    - Example: clear
    - Explanation: Clears the terminal screen.

49. exit
    - Example: exit
    - Explanation: Closes the terminal session.

50. df
    - Example: df -h
    - Explanation: Displays disk space usage of file systems in a human-readable format.
